Step 2: Initialize Project in GitHub

Create folders and files directly on GitHub (or upload manually)

Step 3: Add HelloWorld.java
Create file:

src/main/java/com/example/HelloWorld.java

Code:
package com.example;

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, DevOps CI/CD using GitHub Actions!");
    }
}

Step 4: Add pom.xml
Create file in repo root:

pom.xml

Code:
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0
         http://maven.apache.org/xsd/maven-4.0.0.xsd">

    <modelVersion>4.0.0</modelVersion>
    <groupId>com.example</groupId>
    <artifactId>java-helloworld-ci</artifactId>
    <version>1.0-SNAPSHOT</version>
    <name>HelloWorld Maven App</name>

    <build>
        <plugins>
            <plugin>
                <groupId>org.apache.maven.plugins</groupId>
                <artifactId>maven-compiler-plugin</artifactId>
                <version>3.8.1</version>
                <configuration>
                    <source>17</source>
                    <target>17</target>
                </configuration>
            </plugin>
        </plugins>
    </build>

</project>

Step 5: Add GitHub Actions Workflow
Create file:

.github/workflows/maven-build.yml

Code:
name: Java CI with Maven

on:
  push:
    branches: [ "master" ]

jobs:
  maven:
    runs-on: ubuntu-latest

    steps:
    - name: Checkout Repository
      uses: actions/checkout@v4

    - name: Set up JDK 17
      uses: actions/setup-java@v4
      with:
        java-version: '17'
        distribution: 'temurin'

    - name: Build with Maven
      run: mvn compile

    - name: Run the App
      run: java -cp target/classes com.example.HelloWorld

Step 6: Verify Workflow

Go to Actions tab

Check:

Checkout Repository → Success

Set up JDK → Success

Build with Maven → BUILD SUCCESS

Run the App → Output printed

Step 7: Modify Code to Trigger CI Again

Update HelloWorld.java:

package com.example;

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, DevOps CI/CD using GitHub Actions!");
        System.out.println("Change #1: Verifying automatic workflow trigger on push.");
    }
}
